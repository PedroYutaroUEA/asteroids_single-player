## Importação e Configuração de Venv:

```
> git clone https://github.com/PedroYutaroUEA/asteroids_pygame.git
```

```
> uv venv
```

```
> source .venv/bin/activate.<yourshell>
# must be on project .venv
> which python
```

```
> uv pip install -r requirements.txt
```

### Execução:

```
> python main.py
```
